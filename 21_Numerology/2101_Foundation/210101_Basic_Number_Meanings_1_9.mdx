<h2 id="calculation">Calculation and Analysis Methods</h2>

### Step-by-Step Calculation Process

**Required Information**:
- Full birth date
- Complete name as per official documents

**Calculation Formula**:

1. **Life Path Number**: Add together all digits in your birth date until reducing to a single digit (1-9).
   - Example: Birthdate 23-04-1990 → 2+3+0+4+1+9+9+0 = 28 → 2+8 = 10 → 1+0 = **1**
2. **Expression Number**: Assign values to each letter of your name (A=1, B=2, etc.), sum, and reduce to a single digit.
3. **Soul Urge Number**: Sum the vowels in your name using the same value system, then reduce.

Once calculated, refer to the number meanings for personal insight or Vastu recommendations[1][3][7].
